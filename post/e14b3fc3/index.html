软件/插件/工具/网站 纪录大全

此篇Blog纪录一些实用的软件/插件/网站 etc（长期更新）

  1. 有些使用笔记本办公的用户对于一些软件可以使用Microsoft Store的UMP软件，更加省内存与省电。 - UMP 字样代表同样可在Microsoft Store下载。
  2. 如出现一些网站不能访问，那是你没有翻过“Great Wall”，参考我的另一篇Blog<论梯子是如何搭成的>
  3. 在此承诺不会在我的文章中添加恶意链接，以下链接如出现问题可以直接Google关键词自行搜索，也请留言告知我。
  4. 欢迎留言推荐。

网站

Easyicon
免费的Ico、PNG下载网站

Awesome Wallpapers
一个免费的壁纸网站。（需要梯子）

wikiHow
一个关于各种问题解决方法的网站。

Top Question
专门回答“什么是最好的…”话题的网站。

医学微视
一个医学病症的百科类网站。

异次元
一个推荐软件/教程/电子书…的网站。

ProgramingFont

在线选择喜欢的编程字体。

软件

效率

Quicker
新的效率神器，功能非常强大，可以实现高度DIY-使用右键为每个应用定制快捷功能。你想要的它都有！而且如果有些私人需求可以自己编写脚本或动作，上手成本稍高，但一旦习惯则再也回不去了。

CapsLock+
使用Caps键+其他键实现各种快捷功能，可以自定义，比如在任何情况下可以Caps+Q键搜索、打开文件、软件等，Caps+E/S/D/F实现在编辑是上下左右移动光标，对于笔记本方向键很小的用户很实用。还有很多其他功能。

Ditto - UMP
一款保存多项复制的内容的软件，你可以先复制，然后快捷键唤醒软件面板选择复制的内容，很实用。微软自己win+V键也有类似的功能，不过一重启缓存就没了。

Snipaste - UMP
有名的截图软件。

PNGoo
一款小巧的压缩图片软件。

坚果云
同步文件至云的软件。

QuickLook - UMP
按一下空格快速预览文件内容，而不必打开专用软件，或是等待大型文件加载的时间。

Picgo
图床上传工具。

WGestures
Windows下全局手势控制软件，相信用户用的最多的是左划<-后退。

Microsoft To-Do - UMP
微软的一款Todo应用，在微软商店中搜索安装。

美化

FileTypesMan

一个可以随意替换Win10对应文件格式图标的小工具，美化系统用的。
Mactype
Windows改善字体渲染的软件，必装。（作者已恢复更新）

编辑

Gitnote
一款Markdown编辑器，与Github仓库关联后会自动上传，更实用的是作者集成了一些插件，比如脑图、流程图和番茄钟等，非常值得一用。

Chrome插件

Stylus
为每个网站挑选你喜欢的样式。

The Great Suspender
在你不使用网页的一段时间后，自动暂停网页节省内存。

Darker Reader
一键进入Chrome黑暗模式，使所有网站变成黑底白字，支持快捷键与亮度等调整。（黑夜必备）

wikiWand
美化维基百科页面。

Save Pined Tabs
使用此插件可以使Chrome开启时自动加载你固定在Chrome的标签页，有同步功能。

沙拉查词
非常好用的一款网页查词插件。

Download Plus
一个管理Chrome下载的插件，如何你觉得Chrome查看下载很麻烦，可以使用这个插件。

Vimium
允许你使用快捷键操作网页，比如使用X恢复上一网页，使用T打开新标签页，j-k下滑/上升网页等等，有一点点学习成本，但对于笔记本用户非常值得。

TamperMonkey
最值得你使用的Chrome插件，它可以让你使用别人编写的脚本，通过这些脚本可以实现的功能有：

  1. 复制/下载百度文库页面
  2. 关闭百度的广告
  3. 解析免费观看视频网站付费视频/音乐网站付费下载
  4. 直接下载百度云盘的文件
  5. …

等等功能很多，你想到的想不到的都可以在Greasy-Fork找到（这是查找油猴脚本的一个网站，如有需要还有其他的网站可以自行搜索）。

Imagus
当你刷微博/Twitter或是其他一些需要查看图片的网站，鼠标移动上去即可显示大图。

Bookmark Sidebar
好用的第三方书签插件，替代Chrome自带不好用的书签功能。

Better Scrollbar
嫌Chrome右边的滚动条太粗太碍事了？这个插件可以让它好看一点。

一键管理扩展
此插件可以快速开启/关闭暂时不需要的插件，节省Chrome占用的内存。

Recent History
一个小按钮查看最近浏览历史。

Custom Cursor for Chrome
自定义你的鼠标光标在Chrome的样式。

Checker Plus for Gmail
Gmail必装插件吧，用起来很方便。

Todoist
美观方便使用的Todo-List插件。

替换部分中文字体为雅黑
统一网站中文字体观感，配合Mactype使用更好，有效改善文字发虚问题。

**Font Rendering Enhancer
什么？字体还不够清晰？诺，最后一个弥补字体渲染的插件了。

工具

Speedpan
速盘，一个不限速下载百度云盘里文件的工具。

Dism++
功能非常强大的Windows系统工具，如果你受不了Windows自带应用、自启动项、弹窗或其他，或者你是个强迫症患者，使用此工具究极净化你的Windows系统。
